# SATextField

SATextfield is useful for textfield layout.

There are some of types that use to quick layout design:

* .underLine
* .dashedLine
* .border
* .borderWithCornerRadius
* .password
* .floaty
* .dropdown
* .infoView

## Looks Like

```swift
textfield.type = .underline
```
